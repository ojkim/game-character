# game-character (게임 캐릭터 과제)

## 1. 프로젝트 실행 방법 (Run)

이 프로젝트는 Gradle Wrapper를 사용하여 실행할 수 있습니다.

**Windows (cmd/powershell):**
```bash
./gradlew.bat run
```
**macOS / Linux:**
```bash
./gradlew run
```
## 2. 프로젝트 빌드 방법 (Build)

**Windows (cmd/powershell):**
```bash
./gradlew.bat build
```
**macOS / Linux:**
```bash
./gradlew build
```
